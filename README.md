# 🧮 Counter App

A simple Counter App created for learning purposes. This app demonstrates the basic concept of state management, user interaction, and UI updating using Flutter.

---

## 📌 Features

- ➕ Increment Counter
- ➖ Decrement Counter
- 🔄 Reset Counter
- 🖼️ Simple and user-friendly interface

---


## 🛠️ Built With

- Language: Dart
- Framework: Flutter 
- IDE: VSCode
